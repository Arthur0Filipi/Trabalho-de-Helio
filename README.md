<h4>Nome do App:</h4>
<h5>TrampoJusto</h5>

<h4> Objetivo: <h4/>
<h5>Criar um aplicativo que conecte jovens e trabalhadores a oportunidades de trabalho digno, além de oferecer informações sobre direitos trabalhistas e qualificação profissional<h5/>

<h4>ODS Relacionada: </h4>
<h5>ODS 8 <br/>
Justificativa: escolhemos  a ODS-8 pois ela a aborda as questões socio-economias, um dos grandes problemas do nosso país.</h5>

<h4>Ideia Inicial do Aplicativo:</h4>

<h5>Funcionalidade do App:<br/>
Encontrar vagas de emprego confiáveis<br/>
Avaliar empresas (tipo IFood, mas para trabalho)<br/>
Aprender sobre direitos trabalhistas<br/>
Fazer cursos curtos ou longos para melhorar o currículo<br/>
Destaque para vagas com boas condições (salário justo, contrato, etc.)<br/>
Avaliação de empresas<br/>
Ambiente de trabalho<br/>
Aréa para aprender sobre direitos trabalhistas</h5>

<h5>Publico alvo:<br/>
Trabalhadores e desempregados</h5>

<h5>Diferenciais em relação a soluções existentes:<br/>
O TrampoJusto seria um app que ajuda o usuário a:<br/>
Encontrar vagas de emprego confiáveis<br/>
Avaliar empresas (tipo "Uber/IFood", mas para trabalho)<br/>
Aprender sobre direitos trabalhistas<br/>
Fazer cursos rápidos para melhorar o currículo
</h5>
